# 🚀 Space Design Editor (Sooru AI)

A web-based 3D space design editor built using React, Vite, and Babylon.js.  
This tool allows users to create, edit, and visualize 3D environments interactively.

---

## 🛠️ Tech Stack

- React
- Vite
- Babylon.js
- Zustand (state management)
- Tailwind CSS (UI)

---
